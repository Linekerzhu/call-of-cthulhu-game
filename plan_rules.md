# 🏰 暗影尖塔 (Shadow Tower) 游戏规则设计

> **设计愿景**：火焰纹章的深度战棋 × 杀戮尖塔的卡牌构筑  
> **核心体验**：15分钟快节奏，连招构筑爽点，每局不同的战场  
> **设计理念**：隐藏流派，玩家自己探索"邪路玩法"

## 1. 核心循环

### 1.1 单局流程（15分钟）

```
开始游戏
    ↓ 30秒
选择出身（4选1，影响初始卡组和遗物）
    ↓
第1层（4-5分钟）
├── 普通战斗 ×2
├── 事件 ×1
└── Boss：史莱姆王
    ↓
第2层（5-6分钟）
├── 普通战斗 ×2
├── 精英战 ×1
├── 休息/商店 ×1
└── Boss：暗影巨龙
    ↓
第3层（5-6分钟）
├── 普通战斗 ×1
├── 精英战 ×1
├── 事件 ×1
├── 商店（最后补强）
└── Boss：巫妖王
    ↓
胜利 / 死亡
    ↓
解锁成就、记录排行榜、获得灵魂点
```

### 1.2 地图节点

| 节点 | 出现概率 | 奖励 | 策略价值 |
|------|---------|------|---------|
| **普通战斗** | 40% | 金币10-15 + 卡牌3选1 | 基础资源获取 |
| **精英战斗** | 15% | 金币25-35 + 遗物 + 稀有卡 | 高风险高回报 |
| **商店** | 20% | 购买卡牌/遗物/删卡/回血 | 卡组调整 |
| **休息点** | 15% | 回血30% 或 升级一张卡 | 续航或强化 |
| **事件** | 10% | 随机（正面/负面/选择） | 意外惊喜 |

## 2. 战棋系统

### 2.1 动态地图尺寸

| 战斗类型 | 地图大小 | 地形复杂度 | 预计时长 |
|---------|---------|-----------|---------|
| **普通遭遇** | 5×8 | 简单（平地+少量障碍） | 1-2分钟 |
| **精英战斗** | 7×10 | 中等（地形+高低差） | 2-3分钟 |
| **Boss战** | 10×12 | 复杂（动态变化） | 4-5分钟 |

### 2.2 地形系统

```
┌─────────────────────────────────────────────────────────────┐
│ 地形名称     │ 移动消耗 │ 防御加成 │ 回避加成 │ 特殊效果   │
├─────────────────────────────────────────────────────────────┤
│ 平地 ⬜     │    1     │   +0%    │   +0%    │ 无         │
│ 森林 🌲     │    2     │   +20%   │   +15%   │ 远程命中率-20%│
│ 山地 ⛰️     │    3     │   +30%   │   +10%   │ 无法被击退  │
│ 河流 🌊     │    2     │   +0%    │   +0%    │ 移动力-1   │
│ 城墙 🏰     │    1     │   +40%   │   +0%    │ 占据优势：可俯瞰全场│
│ 火坑 🔥     │    1     │   -10%   │   +0%    │ 停留回合-3HP│
│ 冰面 ❄️     │    1     │   +0%    │   +5%    │ 30%概率滑倒（跳过行动）│
│ 毒沼 ☠️     │    2     │   -20%   │   -10%   │ 停留回合-2HP│
│ 高地 📍     │    2     │   +10%   │   +0%    │ 高地打低地+25%伤害│
└─────────────────────────────────────────────────────────────┘
```

**高地优势详解：**
- 攻击低地目标：伤害+25%，射程+1
- 被低地攻击：伤害-25%
- 远程单位占领高地 = 战略要地

### 2.3 战场动态变化（Boss战）

**【史莱姆王】三阶段战场：**

| 阶段 | HP范围 | 战场变化 | 机制变化 |
|------|--------|---------|---------|
| **阶段1** | 100%-70% | 普通平地 | 史莱姆缓慢移动，单体攻击 |
| **阶段2** | 70%-40% | 地面分裂，出现毒沼（5格） | 分裂成3个小史莱姆，必须同时击杀 |
| **阶段3** | 40%-0% | 毒沼扩散（10格），安全区缩小 | 狂暴状态，10回合内必须击杀否则全屏毒雾 |

**【暗影巨龙】空地一体：**

| 阶段 | 机制 |
|------|------|
| **空中阶段** | 龙在3层高飞，只能远程攻击。地面有火坑陷阱。龙息攻击（直线3格预警） |
| **降落阶段** | 龙降落到地面（1层），可以近战。拍击地面产生冲击波（十字范围击退） |
| **召唤阶段** | 龙飞回2层，召唤3只小龙。玩家需要同时处理小怪和输出Boss |

### 2.4 障碍物系统

**可破坏障碍物：**

| 障碍物 | HP | 破坏奖励 | 战略用途 |
|--------|-----|---------|---------|
| 木箱 📦 | 1 | 金币5-10 | 临时掩体，可挡住远程 |
| 石门 🚪 | 3 | 无 | 阻挡通道，必须破坏或绕路 |
| 栅栏 ⛩️ | 2 | 无 | 阻挡移动，但远程可穿过 |
| 火药桶 🛢️ | 1（火焰引爆） | 范围爆炸伤敌 | 可以主动引爆造成AOE |

## 3. 徽章与职业

### 3.1 徽章系统核心规则

- **单持规则**：同一时间只能装备1个徽章
- **更换规则**：可以在休息点更换徽章，但会失去旧徽章的专属卡
- **获取途径**：精英战必掉、Boss战2选1、特定事件

### 3.2 四大徽章

#### ⚔️ 狂战士徽章

| 属性 | 说明 |
|------|------|
| **核心机制** | **怒气**：受到伤害时获得怒气（每受1伤=1怒气），怒气增加伤害（每点怒气+5%伤害） |
| **初始遗物** | 血怒护符（开局获得10怒气） |
| **初始卡组** | 3张攻击卡 + 2张低费移动卡 |
| **专属卡牌** | 【狂暴打击】【血之渴望】【狂战士之血】【毁灭旋风】 |

**机制详解：**
- 怒气上限：50点（+250%伤害）
- 怒气每回合衰减：-5点
- 鼓励：卖血换输出，高风险高回报

#### 🛡️ 圣骑士徽章

| 属性 | 说明 |
|------|------|
| **核心机制** | **守护**：与友军/幻象相邻时，可以替其承受伤害（50%概率） |
| **初始遗物** | 圣盾纹章（开局获得8点格挡） |
| **初始卡组** | 2张防御卡 + 2张攻击卡 + 1张召唤卡 |
| **专属卡牌** | 【圣盾庇护】【牺牲】【守护誓言】【神圣反击】 |

**机制详解：**
- 守护触发时，圣骑士获得"正义"标记
- 3层正义 → 下张防御卡费用变为0
- 鼓励：保护队友，团队作战

#### 🔮 元素使徽章

| 属性 | 说明 |
|------|------|
| **核心机制** | **元素精通**：每打出3张元素卡，自动释放一次"元素冲击"（随机元素AOE） |
| **初始遗物** | 元素水晶（首回合首张元素卡费用为0） |
| **初始卡组** | 3张元素卡（火/冰/雷各1）+ 2张过牌卡 |
| **专属卡牌** | 【元素共鸣】【魔力溢出】【元素风暴】【元素化身】 |

**机制详解：**
- 元素冲击伤害 = 已打出元素卡数量 × 2
- 鼓励：大量元素卡，快速循环
- 终极形态：一回合多次元素冲击

#### 🏹 游侠徽章

| 属性 | 说明 |
|------|------|
| **核心机制** | **灵动**：每移动2格，抽1张卡（每回合最多3次） |
| **初始遗物** | 风之靴（移动力+1） |
| **初始卡组** | 3张移动相关卡 + 2张远程攻击卡 |
| **专属卡牌** | 【飘逸射击】【疾风步】【穿云箭】【影无踪】 |

**机制详解：**
- 移动包括：主动移动、被击退、传送
- 鼓励：高机动，放风筝战术
- 核心：移动=过牌=资源

### 3.3 隐藏徽章（经典模式解锁）

| 徽章 | 解锁条件 | 核心机制 |
|------|---------|---------|
| ☠️ **死灵法师** | 经典模式通关第1层 | **亡者复生**：击败的敌人有30%概率成为己方骷髅 |
| 🌙 **月影刺客** | 经典模式通关第2层 | **背刺**：从背后攻击伤害×2 |
| ⚡ **风暴领主** | 经典模式通关第3层 | **连锁**：攻击会弹射到相邻敌人 |

## 4. 卡牌系统

### 4.1 卡牌结构

```typescript
interface Card {
  id: string;           // 唯一标识
  name: string;         // 卡名
  cost: number;         // 能量消耗（0-4）
  type: 'attack' | 'defense' | 'skill' | 'power' | 'move';
  rarity: 'basic' | 'common' | 'uncommon' | 'rare';
  keywords: string[];   // 关键词：用于搜索和combo
  description: string;  // 描述文本（无流派标签）
  effects: Effect[];    // 具体效果
}
```

## 5. 连携与支援

### 5.1 相邻连携

**触发条件**：两个单位上下左右相邻

| 连携类型 | 效果 |
|---------|------|
| **攻击连携** | 一方攻击时，另一方追加30%伤害（一次） |
| **防御连携** | 双方获得+15%格挡加成 |

**可以和以下单位形成连携：**
- ✅ 玩家角色
- ✅ 召唤的幻象
- ✅ 召唤的宠物/骷髅
- ✅ 中立NPC（特定事件）

### 5.2 三角包围

**触发条件**：三个单位围住一个敌人（敌人上下左右被占据3格）

| 效果 | 数值 |
|------|------|
| 对被包围敌人伤害 | +50% |
| 被包围敌人无法移动 | 定身 |
| 击败后额外金币 | +20% |

### 5.3 连携相关卡牌

| 卡名 | 费用 | 效果 |
|------|------|------|
| 集结号角 | 2 | 召唤1个幻象在相邻位置（1回合），可触发连携 |
| 战术配合 | 1 | 本回合连携伤害加成变为50% |
| 铁壁阵 | 2 | 获得10点格挡，如果有连携，再获得10点 |
| 合击 | 2 | 造成8点伤害，每有一个连携单位，再造成4点 |

## 6. 熟练度系统

### 6.1 核心规则

- **单局清零**：每局游戏开始时，所有熟练度从E级开始
- **快速成长**：从E到S级约需要打出10-15张同类卡牌
- **视觉反馈**：升级时有明显提示和特效

### 6.2 熟练度等级

| 等级 | 所需卡牌数 | 加成效果 | 解锁内容 |
|------|-----------|---------|---------|
| **E** | 0 | 无 | 基础 |
| **D** | 3张 | 该类卡牌伤害/效果+1 | - |
| **C** | 6张 | 该类卡牌费用-1（最低0） | - |
| **B** | 9张 | 可以装备该类"大师遗物" | 大师遗物解锁 |
| **A** | 12张 | 该类卡牌效果+50% | - |
| **S** | 15张 | 解锁终极卡 | 终极卡加入奖励池 |

### 6.3 终极卡（S级解锁）

| 类型 | 终极卡 | 效果 |
|------|--------|------|
| 剑类 | 【剑圣奥义】 | 3费，造成15点伤害，本回合所有剑卡费用为0 |
| 盾类 | 【绝对防御阵】 | 3费，获得30点格挡，全队无敌1回合 |
| 元素 | 【元素灾变】 | 3费，对所有敌人施加火+冰+雷三重状态 |
| 弓类 | 【箭雨】 | 3费，对全场射10箭，每箭2点伤害 |

## 7. 战斗流程

### 7.1 回合结构

```
【玩家回合】
1. 抽牌阶段：抽5张卡（手牌上限10）
2. 能量恢复：恢复满能量（默认3点）
3. 状态结算：中毒、燃烧等生效
4. 玩家行动：
   - 移动（消耗移动力）
   - 打出卡牌（消耗能量）
   - 使用药水
   - 查看敌人意图
5. 结束回合按钮

【敌人回合】（自动执行）
1. 显示敌人意图（玩家回合已可见）
2. 敌人同时移动
3. 敌人同时执行动作（攻击/防御/buff）
4. 伤害结算（先结算玩家，再结算敌人）
5. 进入下一回合
```

### 7.2 意图可视化

每个敌人头顶显示下回合行动：

| 图标 | 含义 | 数值范围 |
|------|------|---------|
| ⚔️8 | 攻击8点伤害 | 根据敌人类型 |
| 🛡️5 | 获得5点格挡 | 根据敌人类型 |
| ⬆️ | 蓄力（下回合伤害×2） | - |
| ⬇️ | 施加debuff | 显示具体debuff |
| 💥 | 范围攻击 | 显示范围格子 |

**预警系统**：敌人攻击范围格子显示红色高亮

### 7.3 战场掉落

敌人死亡时：

| 掉落类型 | 概率 | 说明 |
|---------|------|------|
| 金币 | 100% | 10-30金币 |
| 武器卡 | 30% | 临时武器卡（本局有效，永久加入卡组） |
| 消耗品 | 25% | 药水、炸弹等 |
| 遗物材料 | 10% | 用于合成（如果有合成系统） |

**临时武器卡示例：**
- 【哥布林短剑】1费，造成5点伤害（获得后本局永久可用）
- 【兽人战斧】2费，造成8点伤害，有击退效果

## 8. Meta系统

### 8.1 出身选择（开局）

| 出身 | 初始卡组 | 初始遗物 | 特殊机制 |
|------|---------|---------|---------|
| **失落的骑士** | 剑类卡×4 + 防御×1 | 破损的剑（剑伤+2） | **荣誉**：不撤退时伤害+20% |
| **被放逐的法师** | 元素卡×3 + 过牌×2 | 元素水晶（首元素卡免费） | **过载**：可透支能量，下回合晕眩 |
| **雇佣兵** | 随机混合×5 | 金币袋（+100金币开局） | **贪婪**：商店-20%，战斗金币-30% |
| **实验体** | 诡异卡×5（各种类型） | 变异细胞（每局随机变一个遗物） | **不稳定**：每场战斗随机buff |

### 8.2 游戏模式

| 模式 | 说明 | 奖励 |
|------|------|------|
| **普通模式** | 可以读档（从本层开始） | 标准奖励 |
| **经典模式** | 永久死亡，无法读档 | 解锁隐藏徽章、排行榜记录 |

### 8.3 解锁系统

**成就解锁：**
- "狂暴之心"：用狂暴流通关 → 解锁新狂暴卡【狂化】
- "铜墙铁壁"：一回合获得50格挡 → 解锁新防御卡【要塞】
- "元素大师"：一回合触发3次元素冲击 → 解锁新元素卡【元素融合】

**徽章解锁：**
- 经典模式通关第1层 → 解锁☠️死灵法师徽章
- 经典模式通关第2层 → 解锁🌙月影刺客徽章
- 经典模式通关第3层 → 解锁⚡风暴领主徽章

### 8.4 灵魂点（长期进度）

- 每局结束获得灵魂点（根据表现）
- 灵魂点用于：
  - 永久解锁某张卡（让它出现在初始奖励池）
  - 升级初始遗物
  - 解锁新出身

## 附录：设计原则总结

1. **隐藏流派**：不标注流派，玩家自己探索combo
2. **快速成长**：熟练度单局清零但升级快，每局都有成长感
3. **战棋深度**：火焰纹章式地形+高低差+连携
4. **15分钟节奏**：快速战斗，密集爽点
5. **构筑自由**：出身只是起点，完全自由发展
6. **经典挑战**：永久死亡模式满足硬核玩家
