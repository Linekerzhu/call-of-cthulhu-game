## 🐙 克苏鲁化改造内容

### 1. 全新卡牌系统 - 深渊的力量

**基础卡调查员求生技能**:
- `疯狂挥击` - 在绝望中挥舞武器
- `理智屏障` - 用残存理智构筑护盾
- `逃离深渊` - 逃离不可名状的恐怖
- `古神低语` - 聆听深渊的治愈低语
- `疯狂之源` - 从疯狂中汲取力量
- `献祭诱饵` - 引诱敌人攻击

**深渊使者徽章卡** (原暴躁鸭):
- `触手蔓延` - 触手缠绕增伤
- `疯狂鞭挞` - 触手疯狂抽打
- `深渊凝视` - 古神注视，攻击卡0费
- `不可名状之击` - 召唤深渊之力
- `理智献祭` - 失去HP获得能量
- `疯狂漩涡` - 消耗疯狂值AOE

**旧日支配者徽章卡** (原铁壳龟):
- `古老护盾` - 古老符文护盾
- `符文反击` - 护盾转攻击
- `深渊反弹` - 古老诅咒反弹
- `绝对屏障` - 旧日支配者防御
- `深渊庇护` - 为同伴提供庇护

**新增黄衣信徒徽章** (控制型):
- `黄衣之王的印记` - 诅咒标记
- `死灵之书残页` - 抽卡失去理智
- `疯狂漩涡`

**特殊诅咒卡**:
- `克苏鲁的呼唤` - 20/30点伤害
- `黄衣之王的印记` - 伤害翻倍标记
- `死灵之书残页` - 抽卡失理智

### 2. 全新徽章系统

| 徽章 | 图标 | 类型 | 被动技能 |
|------|------|------|----------|
| **深渊使者** | 🐙 | 攻击型 | 疯狂值系统：10点疯狂值时攻击翻倍并获得触手缠绕 |
| **旧日支配者** | 👁️ | 防御型 | 古老符文：格挡8点下回合+2格挡并反弹2伤害 |
| **黄衣信徒** | 🎭 | 控制型 | 疯狂诅咒：敌人20%几率恐惧跳过行动 |

### 3. 全新敌人 - 克苏鲁神话怪物

**Boss**: 
- `深渊领主` 🐙 - 克苏鲁化身，100HP，12攻，2行动，召唤触手

**精英敌人**:
- `深潜者祭司` 🧜 - 35HP，法术射程2
- `修格斯` 🦠 - 60HP，再生3HP/回合
- `星之眷族` 👁️ - 30HP，攻击减少玩家理智

**普通敌人**:
- `深潜者` 🐟 - 被诅咒的鱼人
- `食尸鬼` 💀 - 墓地不死生物
- `古老者` 🦑 - 远古文明种族
- `夜魇` 🦇 - 深渊飞行怪物

### 4. 克苏鲁视觉主题

**颜色方案**:
- 主色调：深血红色 (#8B0000)
- 次要色：病态海绿色 (#2E8B57)
- 强调色：暗紫色 (#9400D3)
- 背景：深海黑 (#0a0a1a) + 深渊紫 (#1a1a2e)

**新增CSS文件**: `cthulhu-theme.css`
- 古神召唤发光动画
- 疯狂脉冲效果
- 触手攻击范围样式
- 血迹伤害数字
- 古老符文高亮
- 异界青攻击范围

**字体**: Cinzel (神秘风格) + Noto Serif SC

### 5. 理智值(SAN值)系统

**新增属性**:
```javascript
sanity: 50,      // 当前理智值
maxSanity: 50,   // 最大理智值
madness: 0       // 疯狂值（深渊使者使用）
```

**UI显示**:
- 战斗界面：🧠 理智值条
- 地图界面：🧠 理智值显示
- 低理智警告：<30%时粉色闪烁

**理智值效果**:
- 理智值低时可能出现幻觉
- 某些敌人攻击减少理智
- 卡牌效果可能消耗理智

## 🎮 游戏机制变化

### 理智值系统
- 受到古神攻击时失去理智
- 理智过低会影响判断（未来可扩展）
- 休息站可恢复理智

### 疯狂值系统（深渊使者）
- 造成伤害积累疯狂值
- 10点疯狂值触发攻击翻倍
- 视觉特效：触手缠绕

### 敌人特殊能力
- **深渊领主**: 召唤触手
- **修格斯**: 再生HP
- **星之眷族**: 理智吸取
- **深潜者祭司**: 法术射程
