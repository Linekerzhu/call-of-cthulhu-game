## 🔧 修改内容

### 1. 敌人属性系统

为每个敌人添加了三个核心属性：

| 属性 | 说明 | 普通怪 | 精英 | Boss |
|------|------|--------|------|------|
| **strength** 💪 | 力量（攻击力） | 4 | 7 | 10 |
| **speed** 👟 | 速度（移动力） | 1 | 2 | 1 |
| **actions** ⚡ | 行动值（每回合行动次数） | 1 | 1 | 2 |

**战斗策略变化**:
- 精英敌人移动更快（2格），需要提前规划位置
- Boss有2次行动，可以在一回合内移动+攻击或连续攻击
- 不同敌人需要不同的应对策略

### 2. 格挡机制优化

**修改前**:
- 玩家结束回合时清零格挡
- 格挡无法抵挡敌人攻击

**修改后**:
- 格挡值保留到敌人回合结束
- 玩家回合开始时清零格挡
- 格挡可以正常抵挡敌人攻击

**代码变更**:
```javascript
// CombatSystem.js - endTurn()
- player.block = 0;  // 删除这行

// CombatSystem.js - startTurn()
+ if (player.block > 0) {
+     this.game.log('格挡值清零');
+     player.block = 0;
+ }
```

### 3. 敌人AI改进

**新移动算法** `moveEnemyWithSpeed()`:
- 敌人根据 `speed` 属性移动多格
- 每格移动会检查是否进入攻击范围
- 进入范围后停止移动，转为攻击

**行动逻辑**:
```javascript
while (敌人还有行动次数) {
    if (在攻击范围内) {
        攻击玩家;
        消耗行动次数;
    } else {
        根据速度移动;
        消耗行动次数;
    }
}
```

## 🎯 游戏平衡性

**当前数值**:
- 普通敌人：12HP，4攻，1速，1行动
- 精英敌人：35HP，7攻，2速，1行动  
- Boss：80HP，10攻，1速，2行动

**策略要点**:
1. 精英敌人速度快，需要保持距离或提前准备格挡
2. Boss有2次行动，一回合可以移动+攻击，威胁更大
3. 格挡现在可以有效抵挡敌人攻击，防御型玩法更可行
4. 铁壳龟徽章的格挡收益更高
