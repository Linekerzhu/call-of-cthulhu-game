# 🏰 暗影尖塔 - 最终版开发计划（方案A）

> **版本**：方案A - 质量优先版  
> **工期**：21天  
> **目标**：高质量MVP，可发布版本  
> **核心原则**：2徽章、2层地图、30张卡、8敌人2Boss  

---

## 📋 最终确定的MVP范围

### ✅ 保留功能

| 系统 | 内容 |
|------|------|
| **徽章** | 暴躁鸭（怒气）+ 铁壳龟（守护） |
| **卡牌** | 30张（6基础 + 12暴躁鸭 + 12铁壳龟） |
| **敌人** | 8种普通 + 2精英 + 2Boss |
| **地图** | 2层（第3层v1.1添加） |
| **地形** | 3种（平地/森林/高地） |
| **连携** | 相邻+30%伤害 |
| **商店** | 买卡/删卡/回血 |
| **休息** | 回血30% 或 升级1卡 |

### ❌ 延后到v1.1

- 泡泡鱼徽章
- 跳跳兔徽章  
- 第3层地图
- 喷火娃Boss
- 事件系统
- 经典模式
- 排行榜联网
