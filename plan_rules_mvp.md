# 🏰 暗影尖塔 MVP规则文档

> **版本**：MVP v1.0  
> **目标**：最小可行产品，验证核心循环  
> **时长**：15分钟/局  
> **核心**：徽章系统 + 卡牌构筑 + 战棋战斗

## 📋 保留的核心系统

### 1. 徽章系统（4个）
- 单持规则，休息点可更换
- 每个徽章有4张专属卡

### 2. 卡牌系统（30张）
- 攻击/防御/技能/移动
- 升级系统保留

### 3. 战棋系统
- 3种地形：平地、森林、高地
- 5×8 / 7×10 / 10×12 动态地图
- 意图可视化

### 4. 资源系统
- 统一资源：**能量**（移动+打牌都用）
- 每回合3点能量
- 移动1格 = 1能量

### 5. Meta系统
- 出身选择（4个）
- 地图节点（战斗/商店/休息/事件/Boss）
- 解锁系统（成就解锁新卡）

## ⚔️ 战斗系统（MVP版）

### 资源规则

```
每回合：
- 抽5张卡
- 恢复3点能量
- 手牌上限10张

能量用途：
- 移动1格 = 1能量
- 打出卡牌 = 卡面费用

示例：
有3能量，可以：
- 移动3格
- 移动1格 + 打出2费卡
- 打出3张1费卡
```

### 地形系统（MVP简化版）

| 地形 | Emoji | 效果 | 说明 |
|------|-------|------|------|
| **平地** | ⬜ | 无效果 | 默认地形 |
| **森林** | 🌲 | +20%防御 | 适合防守 |
| **高地** | 📍 | 伤害+25% | 战略要地 |

### 连携系统（MVP简化版）

```
相邻加成：
- 玩家与召唤物相邻 → 伤害+30%
- 删除三角包围（MVP太复杂）

召唤物：
- 【大喇叭】召唤的纸片人（持续3回合）
```

### 意图可视化

```
敌人头顶显示：
⚔️8 → 下回合攻击8点伤害
🛡️5 → 下回合获得5点格挡
⬆️ → 蓄力（下回合伤害×2）

攻击范围格子红色高亮预警
```

## 🗺️ 地图系统（MVP版）

### 单局流程

```
第1层（5分钟）
├── 普通战斗 ×2
├── 事件 ×1
└── Boss：大果冻

第2层（5分钟）
├── 普通战斗 ×2
├── 精英战 ×1
├── 商店/休息 ×1
└── Boss：喷火娃

第3层（5分钟）
├── 普通战斗 ×1
├── 精英战 ×1
├── 事件 ×1
├── 商店
└── Boss：老不死
```

### 节点类型

| 节点 | 奖励 |
|------|------|
| **战斗** | 零花钱10-15 + 卡牌3选1 |
| **精英** | 零花钱25-35 + 遗物 + 稀有卡 |
| **商店** | 买卡/遗物/删卡/回血 |
| **休息** | 回血30% 或 升级1张卡 |
| **事件** | 随机好坏 |

## ⚙️ 技术规格（MVP约束）

### 性能目标

```
- 首屏加载 < 3秒（4G网络）
- 帧率稳定 30fps
- 内存占用 < 100MB
- 支持 iPhone 8 及以上
```

### 代码约束

```
- 单HTML文件（像现在一样）
- 无框架依赖（原生JS）
- 总大小 < 500KB
- 可使用 PixiJS（轻量级）
```

### 响应式断点

```css
/* 只支持两种布局 */
/* 手机竖屏 */
@media (max-width: 767px) { ... }

/* 平板/桌面 */
@media (min-width: 768px) { ... }
```
